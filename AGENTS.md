# AGENTS.md

## Goal
Build a simple and maintainable C# desktop application.

## Rules

- Use standard .NET project structure
- Keep dependencies minimal
- Prefer WPF or WinForms unless specified
- Ensure buildability

## Code Guidelines

- Keep classes small
- Separate UI and logic
- Avoid over-engineering
- Add tests if applicable

## Output Requirements

Always include:
- changed files
- summary of changes
- reason for changes
- build considerations
- next steps

## Workflow

1. Analyze repository
2. Propose plan
3. Implement minimal working version
4. Ensure build passes
5. Update README if needed
